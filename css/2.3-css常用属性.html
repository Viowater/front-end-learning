这是一个块级元素
这是一个行内元素,不受width和height影响

img标签是一种行内块元素, 即行内的块元素,特性可想而知

可以通过display属性修改元素类型
块级元素就是block;行内就是inline;行内块级就是inline-block

内容(content):盒子包含的实际内容,如文本,图片等

内边距(padding):内容与边框的间距

边框(border):无需多言

外边距(margin):盒子与其他元素之间的空间

a border example, border也是复合属性
border-width/style/color都是复合属性, 按顺时针从上开始获取width,面对面的位如果没指定,自动获取对面的值(比如只填三个,缺少右值,右copy左)

指定单个方向的边框(以左为例)就border-left然后填width style color值即可(顺序固定,只能从后省略)
也可以border-left-具体 来指定某个
